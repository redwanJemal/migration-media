## Content Voice
- Authoritative but approachable
- Data-driven (always include numbers)
- Bilingual: English primary, Amharic for Ethiopian-targeted content
- Action-oriented ("Apply NOW", "Save this", "Tag someone")
- Always legal — never promote illegal migration

## Thumbnail Style
- 1080x1080 (Instagram/Telegram) — generated automatically
- 1920x1080 (YouTube) — TODO: create landscape template
- Bold uppercase headlines
- Country flag emoji for visual anchoring
- Highlight box with key numbers
- "MigrationPath" watermark bottom-left
- Deadline indicator when applicable

## Content Rules
1. Every post MUST include a specific action (link, deadline, next step)
2. Every scholarship post MUST state if it's fully funded or partial
3. Never promise guaranteed outcomes
4. Always mention "legal" migration paths
5. Include Amharic caption for Ethiopia-targeted content
6. End with engagement prompt (question, tag request, save request)
